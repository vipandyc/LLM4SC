# Ablation 1 — Paraphrase Few-Shot Examples

## Goal

Test whether paraphrasing the 23 few-shot examples in `prompt/prompt_fewshot_distilled.md`
(keeping scientific content identical) changes the opinion score distributions.

## Setup

- **Sample**: 50 papers drawn from `data/SC_related_cites_010.csv` (random seed 42)
- **Model**: `gpt-5-mini`
- **Baseline prompt**: `prompt/prompt_fewshot_distilled.md`
- **Paraphrased prompt**: `prompt/prompt_fewshot_paraphrased.md` (generated by `generate_paraphrased_prompt.py`)

## Scripts

| Script | Description |
|---|---|
| `generate_paraphrased_prompt.py` | Calls GPT to paraphrase each of the 23 examples; saves to `prompt_fewshot_paraphrased.md` |
| `run_ablation1.py` | Scores the 50-paper sample with both prompts |
| `compare_results.py` | Computes per-mechanism mean/std, Spearman ρ, MAD; saves plots |

## Results (n = 50)

| Mechanism | Baseline mean | Para mean | Mean abs diff | Spearman ρ |
|---|---|---|---|---|
| El-ph | 0.60 | 0.46 | 0.22 | 0.749 |
| Bipolaron | 0.00 | 0.00 | 0.00 | N/A |
| AFM | 0.62 | 0.58 | 0.12 | 0.913 |
| FM | 0.34 | 0.36 | 0.02 | 0.919 |
| CDW | 0.44 | 0.42 | 0.02 | 1.000 |
| Nematic | 0.18 | 0.12 | 0.06 | 0.700 |
| Plasmon | 0.00 | 0.00 | 0.00 | N/A |
| Correlation | 0.48 | 0.38 | 0.10 | 0.935 |
| Spin liquid | 0.10 | 0.12 | 0.02 | 0.714 |

Parse failures: Baseline 0/50, Paraphrased 0/50.

## Interpretation

- **High Spearman correlations** (ρ ≥ 0.7 for all non-trivial mechanisms) indicate the model produces
  consistent relative rankings regardless of example phrasing.
- **Mean absolute differences are small** (≤ 0.22 across all mechanisms), suggesting score distributions
  are robust to paraphrase variation in the few-shot examples.
- The slight drop in El-ph mean (0.60 → 0.46, MAD = 0.22) is the largest shift observed, suggesting
  El-ph scoring has marginally more sensitivity to example wording than other mechanisms.
- **Conclusion**: The baseline few-shot examples are not a major source of variance; paraphrasing
  does not substantially alter the score distributions.
